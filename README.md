# ABYSS // depth-log

A brutalist, scientific archive of the world's pelagic zones and the creatures that hold the dark together.

## Files
- `index.html` — Hero, depth scale, five pelagic zones, twelve specimen cards (each with custom inline SVG), telemetry data table
- `specimen.html` — Detailed specimen page for Anglerfish (illustrated cross-section + long-form essay)
- `styles.css` — Full design system
- `main.js` — Scroll-mapped depth indicator (0m → −10,994m Challenger Deep)

## Aesthetic
- Off-black ocean palette + bone foreground + cyan/coral/amber accents
- IBM Plex Mono + Plex Sans + Instrument Serif (italic display)
- CSS scanline overlay for screen texture
- Custom SVG silhouettes for every specimen
- Sticky depth ruler on the left, scroll-depth indicator on the right

## Deploy to GitHub Pages
1. Push to a public repo
2. Settings → Pages → Source: deploy from `main`, root `/`
3. Live at `https://<user>.github.io/<repo>/`
